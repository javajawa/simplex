Simplex Solver

Quick Intro

Welcome to the simplex solver. Input your problem below, and click 'Solve...' to solve the problem by choosing pivots manually, or 'Auto Solve' to have the script maximise the values.

The input form only shows the actual variables — slack variables are generated automatically. Numbers can be inserted in the form '1.234' or '12/5'. All numbers will be converted to minimal rationals at all stages.

The first row should contain your objective function: the value you are trying to maximise. This will later be assigned to x0. Remaining rows are your constraints.

The constraints should be entered as less-than-or-equals to expressions with all variables being <= a constant; for greater-than expressions, simply change the sign of every term. The auto solver does not currently support equal-to expressions.

The source code for this tool can be found on github.
